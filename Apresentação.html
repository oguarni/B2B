Objetivo do Projeto

Desenvolver um MVP (Minimum Viable Product) de uma plataforma de e-commerce que demonstre a aplicação de conceitos de engenharia de software na construção de um sistema de vendas online, integrando frontend, backend e banco de dados, com foco em segurança e usabilidade.

Tema Escolhido: E-commerce B2B para Indústrias

A escolha se deu pela complexidade e relevância do mercado B2B, permitindo explorar funcionalidades avançadas como sistemas de cotação, múltiplos perfis de usuário e validações específicas, indo além do escopo de um e-commerce B2C tradicional.

Arquitetura e Tecnologias

Frontend

Interface reativa construída com React.js para uma experiência de usuário dinâmica e componentizada. Estilização moderna e responsiva com Tailwind CSS.

Backend

API RESTful robusta desenvolvida em Node.js com Express.js. Interação com o banco de dados gerenciada pelo ORM Sequelize para abstração e segurança.

Banco de Dados e Infra

Utilização do PostgreSQL, um banco de dados relacional poderoso. O ambiente de desenvolvimento e produção é containerizado com Docker e Docker Compose para garantir consistência e facilitar o deploy.

Atendimento aos Requisitos Funcionais

✓ Cadastro/Login

Registro de usuários e empresas com autenticação via JWT.

✓ Catálogo

Listagem de produtos, imagens, preços e filtros por categoria.

✓ Carrinho (Adaptado)

Substituído por um "Pedido de Cotação", mais adequado ao modelo B2B.

✓ Checkout

Formulário de pagamento simulado via PIX, gerando um pedido formal.

✓ Admin

Painel para CRUD completo de produtos, acessível a usuários com perfil "admin".

★ Feature Extra

Sistema de comparação de cotações, permitindo ao comprador analisar múltiplas propostas.

Atendimento aos Requisitos Não-Funcionais

Segurança

  • Senhas dos usuários são criptografadas no banco de dados com Bcrypt.js.
  • Rotas protegidas e controle de acesso baseado em perfis (roles) utilizando JSON Web Tokens (JWT).
  • Uso do ORM Sequelize para mitigar riscos de ataques de SQL Injection.

Usabilidade e Performance

  • Interface totalmente responsiva (Mobile First) desenvolvida com Tailwind CSS.
  • Feedback visual constante para o usuário através de toasts, modais e estados de carregamento.
  • Componentes lazy-loaded no React para otimizar o tempo de carregamento inicial (TBT).

Conclusão

O projeto ConexHub cumpre com sucesso todos os requisitos funcionais e não-funcionais especificados, entregando um MVP robusto e bem arquitetado.
